# Showcase Fixtures

This directory contains stable input data for showcase programs.

- `sample_weather.txt` is the default text fixture for `bench_02_read_stats`
  and `bench_03_find_count`.
- `tree_fixture/` is the directory fixture for `bench_01_tree` and
  `bench_08_path_list_helpers`.

These files are intended for smoke runs, scripts, and deterministic e2e tests.
