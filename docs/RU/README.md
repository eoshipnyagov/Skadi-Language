# RU Release Docs

This directory is reserved for Russian release snapshots.

Rules:
- Day-to-day editing happens in English docs under `docs/`.
- Before release, required docs are duplicated here in Russian.
- Each RU file must contain:
  - `Source: docs/<EN_FILE>.md`
  - `Synced for release: <version/date>`

See policy: `docs/DOCS_POLICY.md`.
