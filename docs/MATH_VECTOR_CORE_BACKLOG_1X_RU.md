# Math/Vector Core Backlog (отложено до 1.x)

Статус: отложено до 1.x
Дата: 2026-05-26

## Зачем это нужно

Для gamedev/embedded ядру нужны удобные и предсказуемые math/vector примитивы, но они не блокируют релиз базового v1 транспилятора.

## Кандидаты в ядро 1.x

- Константы: `PI`, `E`, `TAU`, `EPSILON`.
- Базовая математика: `sin`, `cos`, `atan2`, `sqrt`, `pow`, `abs`, `min`, `max`, `clamp`.
- Утилиты чисел: `round`, `floor`, `ceil`, `trunc`, `fract`.
- Нормализация угла: `normalize_angle`.
- RNG baseline: `rand()` и детерминированный `seed(...)`.
- Векторные типы: `Vec2`, `Vec3`, `Vec4` (арифметика, `dot`, длина, normalize).
- Обертки для 2D-матриц поверх `List` (без отдельного тяжелого runtime на первом шаге).

## Что уже сейчас

- Часть математических builtin объявлена как design-target,
- но полноценный runtime/API для этого блока пока не включен в stable v1.
