# Math/Vector Core Backlog (отложено до 1.x)

Статус: отложено до 1.x  
Дата: 2026-05-22  
Контекст: приоритет для направления gamedev/embedded, но не блокирует текущий этап стабилизации компилятора.

## 1. Константы

- `PI`
  - Что это: число pi (`3.141592...`), отношение длины окружности к диаметру.
  - Где применяется: углы, тригонометрия, физика, анимации, повороты.
  - Частота: **очень часто**.

- `TAU`
  - Что это: `2 * PI` (полный оборот в радианах, примерно `6.28318`).
  - Где применяется: циклические эффекты, фазовые расчеты, повороты “полным кругом”.
  - Частота: **средне** (часто удобнее `PI`, но для некоторых задач `TAU` чище).

- `E`
  - Что это: основание натурального логарифма (`2.71828...`).
  - Где применяется: экспоненты, фильтры, затухания, матмодели.
  - Частота: **средне**.

- `EPSILON`
  - Что это: маленькое число-точность для сравнений float (например, `1e-6`).
  - Где применяется: сравнение чисел с плавающей точкой (`abs(a-b) < EPSILON`).
  - Частота: **очень часто** в math/physics/graphics.

## 2. Базовые численные функции

- `abs`, `sign`
  - Где применяется: физика, обработка сигналов, контроллеры.
  - Частота: **очень часто**.

- `min`, `max`, `clamp`
  - Где применяется: ограничения параметров, UI, физика, безопасные диапазоны.
  - Частота: **очень часто**.

- `floor`, `ceil`, `round`
  - Где применяется: пиксельные координаты, дискретизация, UI layout.
  - Частота: **часто**.

- `trunc`
  - Что это: отбрасывает дробную часть без округления.
  - Пример: `trunc(3.9)=3`, `trunc(-3.9)=-3`.
  - Где применяется: быстрый переход float->int-подобному значению.
  - Частота: **средне**.

- `fract`
  - Что это: дробная часть числа.
  - Пример: `fract(3.9)=0.9`.
  - Где применяется: процедурная графика, тайлинг, фазовые эффекты.
  - Частота: **средне/часто** в gamedev.

## 3. Тригонометрия и углы

- `sin`, `cos`, `tan`
  - Частота: **очень часто**.

- `asin`, `acos`, `atan`, `atan2`
  - `atan2(y, x)`:
  - Что это: угол вектора `(x, y)` с корректным учетом квадранта.
  - Где применяется: наведение, поворот к цели, 2D-навигация.
  - Частота: **очень часто** в играх/роботике.

- `deg_to_rad`, `rad_to_deg`
  - Частота: **часто** (конверсия между удобными человеку градусами и рабочими радианами).

- `normalize_angle`
  - Что это: приведение угла к стандартному диапазону, например:
  - `[-PI, PI)` или `[0, TAU)`.
  - Где применяется: чтобы угол “не убегал” при накоплении и корректно считались разницы.
  - Частота: **часто** в камерах, AI, управлении, физике вращений.

## 4. Степени, корни, логи

- `sqrt`
  - Частота: **очень часто**.

- `root(x, n)`
  - Что это: корень степени `n` (например кубический корень при `n=3`).
  - Частота: **средне**.

- `pow`, `exp`, `ln`, `log2`, `log10`
  - Частота: **средне** (часто для процедурных систем и матмоделей).

## 5. Интерполяция/плавность

- `lerp`, `inverse_lerp`, `remap`
  - Частота: **очень часто** в геймплее, анимации, UI, аудио.

- `smoothstep`, `smootherstep`
  - Частота: **часто** для мягких переходов.

## 6. Случайность

- `rand()`
  - Частота: **очень часто**.

- `rand_range(min, max)` (int/float)
  - Частота: **очень часто**.

- `rand_seed(seed)`
  - Что это: инициализация генератора для воспроизводимости.
  - Частота: **очень часто** для отладки и deterministic replay.

## 7. Векторы

- Встроенные типы: `Vec2`, `Vec3`, `Vec4`
  - Частота: **очень часто** (особенно `Vec2`/`Vec3`).

- Операции: `+ - * /`, scalar ops
  - Частота: **очень часто**.

- `dot`, `length`, `length_sq`, `normalize`, `distance`, `distance_sq`
  - Частота: **очень часто**.

- `cross` (для `Vec3`)
  - Частота: **часто** в 3D.

## 8. Матрицы (2D-фокус)

- Цель: удобный синтаксис 2D-матриц (как оболочка над `List`/типом `Matrix2D`).
- Базовые операции: `identity`, `mul`, `transpose`, `transform_point`.
- 2D-трансформы: `translate`, `rotate`, `scale`.
- Частота: **часто** для рендера/камер/координатных преобразований.

## 9. Embedded-приоритеты

- Явные `f32`-варианты math API.
- Предсказуемость и минимальный runtime overhead.
- Deterministic поведение для random/math, где это критично.
